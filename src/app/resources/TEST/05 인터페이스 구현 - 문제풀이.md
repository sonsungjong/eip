## 📝 REST / SOAP

**Q4.** 인터페이스 구현 기술인 REST와 SOAP에 대한 설명이다. 괄호에 알맞은 말을 쓰시오.

* ( ① ) : HTTP, HTTPS 등을 통해 XML 기반의 메시지를 교환하는 프로토콜. WSDL과 UDDI를 사용하며 비교적 무겁다.
* ( ② ) : HTTP 프로토콜의 메서드(GET, POST, PUT, DELETE)를 그대로 활용. 자원(Resource)을 URI로 표현하며 가볍다.

<details>
<summary>정답 보기</summary>

① **SOAP** (Simple Object Access Protocol)
② **REST** (Representational State Transfer)

</details>
