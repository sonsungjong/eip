## 📝 정규화 (Normalization)

**Q5.** 정규화 과정을 1NF부터 5NF까지 순서대로 쓰고, 각 단계의 조건을 쓰시오. *(암기: 도·부·이·결·다·조)*

<details>
<summary>정답 보기</summary>

| 단계 | 조건 |
|:---|:---|
| **1NF** | **도**메인이 원자값 |
| **2NF** | **부**분 함수 종속 제거 |
| **3NF** | **이**행적 함수 종속 제거 |
| **BCNF** | **결**정자가 모두 후보키 |
| **4NF** | **다**중값 종속 제거 |
| **5NF** | **조**인 종속성 이용 |

</details>

## 📝 트랜잭션 & 병행 제어

**Q13.** 트랜잭션의 4가지 특성(ACID)을 쓰고 각각의 의미를 간략히 설명하시오.

<details>
<summary>정답 보기</summary>

| 특성 | 설명 |
|:---|:---|
| **Atomicity (원자성)** | 모두 반영되거나 전혀 반영되지 않아야 함 (All or Nothing) |
| **Consistency (일관성)** | 완료 후 언제나 일관된 DB 상태를 유지 |
| **Isolation (독립성)** | 동시 실행 시 서로의 연산에 끼어들 수 없음 |
| **Durability (영속성)** | 완료된 결과는 시스템 장애에도 영구 반영 |

</details>
