## 2. 연산자 (Operators)

### 산술 연산자
* `+`, `-`, `*`, `/` (몫, 정수끼리 나누면 정수), `%` (나머지)

### 증감 연산자 ★빈출★
* `++x` (전위): **먼저 1 증가** 후 연산에 사용
* `x++` (후위): 연산에 사용 후 **나중에 1 증가**

```c
int a = 5;
printf("%d", a++);  // 출력: 5 (출력 후 a는 6)
printf("%d", ++a);  // 출력: 7 (먼저 증가 후 출력)
```

### 비트 연산자

| 연산자 | 이름 | 설명 |
| :--- | :--- | :--- |
| `&` | AND | 두 비트 모두 1이면 1 |
| `\|` | OR | 하나라도 1이면 1 |
| `^` | XOR | 서로 다르면 1 |
| `~` | NOT | 비트 반전 |
| `<<` | 좌 시프트 | 비트를 왼쪽으로 이동 (×2 효과) |
| `>>` | 우 시프트 | 비트를 오른쪽으로 이동 (÷2 효과) |

### 논리 연산자
* `&&` (AND), `||` (OR), `!` (NOT)

### 삼항 연산자
```c
결과 = (조건) ? 참일_때_값 : 거짓일_때_값;
```

### 연산자 우선순위 (높은 순) ★빈출★
```
괄호() → 단항(++, --, !, ~) → 산술(*, /, %) → 산술(+, -)
→ 시프트(<<, >>) → 관계(<, >, <=, >=) → 관계(==, !=)
→ 비트(&) → 비트(^) → 비트(|)
→ 논리(&&) → 논리(||) → 삼항(?:) → 대입(=, +=, -=)
```

> **핵심:** 산술 > 시프트 > 관계 > 비트 > 논리 > 대입

## 3. C언어 출력 (printf) 서식 지정자 ★빈출★

| 서식 | 설명 | 예시 |
| :--- | :--- | :--- |
| **%d** | 10진 정수 | printf("%d", 10); → 10 |

| **%x** | 16진 정수 (소문자) | printf("%x", 255); → ff |

```c

printf("%.2f", 3.14); // "3.14" (소수점 2자리)
```

## 4. 포인터와 배열 (C언어) ★매우 중요★

### 포인터 기본
```c
int num = 10;
int *ptr = &num;    // ptr은 num의 주소를 저장
printf("%d", *ptr); // 출력: 10 (ptr이 가리키는 값)
printf("%p", ptr);  // 출력: num의 메모리 주소
```

### 배열과 포인터 관계 ★매우 빈출★
```c
int arr[5] = {10, 20, 30, 40, 50};
// 배열명 arr은 배열의 시작 주소 (== &arr[0])
// arr[i] == *(arr + i) (동일한 표현)

printf("%d", arr[2]);      // 30
printf("%d", *(arr + 2));  // 30 (동일)
printf("%d", *(arr));      // 10 (첫 번째 요소)
```

### 2차원 배열
```c
int arr[2][3] = {{1,2,3}, {4,5,6}};
printf("%d", arr[1][2]);  // 6
```

### 문자열 (char 배열)
```c
char str[] = "Hello";
printf("%s", str);     // Hello
printf("%c", str[0]);  // H
printf("%c", *(str+1)); // e
```

### 구조체 포인터
```c
struct Student { char name[20]; int age; };
struct Student s = {"Kim", 20};
struct Student *ptr = &s;
printf("%s", ptr->name);  // Kim (-> 연산자)
printf("%d", (*ptr).age); // 20 (동일)
```

### 동적 메모리 할당
```c
int *p = (int*)malloc(sizeof(int) * 5);  // int 5개 크기 할당
free(p);  // 반드시 해제 (메모리 누수 방지)
```

## 5. 제어문과 반복문

### 조건문
```c
if (조건1) { ... }
else if (조건2) { ... }
else { ... }

switch (변수) {
    case 값1: ...; break;
    case 값2: ...; break;
    default: ...;
}
// break가 없으면 다음 case로 fall-through (계속 실행)
```

### 반복문
```c
for (초기식; 조건식; 증감식) { ... }

while (조건식) { ... }

do { ... } while (조건식);  // 최소 1번 실행 보장
```

### break와 continue
* **break:** 반복문을 즉시 탈출
* **continue:** 현재 반복을 건너뛰고 다음 반복으로 이동

### 중첩 반복문 예시
```c
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 2; j++) {
        printf("%d%d ", i, j);
    }
}
// 출력: 00 01 10 11 20 21
```

## 6. 재귀함수 ★매회 출제★
자기 자신을 호출하는 함수입니다. 호출 스택을 추적하여 결과를 구하는 문제가 자주 나옵니다.

```c
int factorial(int n) {
    if (n <= 1) return 1;        // 종료 조건 (Base Case)
    return n * factorial(n - 1);  // 재귀 호출
}
// factorial(5) = 5 * 4 * 3 * 2 * 1 = 120
```

### 피보나치 수열 (재귀)
```c
int fib(int n) {
    if (n <= 1) return n;
    return fib(n-1) + fib(n-2);
}
// fib(5) = fib(4)+fib(3) = 3+2 = 5
```

## 7. 객체지향 프로그래밍 (Java)

### 접근 제어자 ★빈출★

| 접근 제어자 | 같은 클래스 | 같은 패키지 | 하위 클래스 | 외부 |
| :--- | :---: | :---: | :---: | :---: |
| **public** | O | O | O | O |
| **protected** | O | O | O | X |
| **default (생략)** | O | O | X | X |
| **private** | O | X | X | X |

> **접근 범위:** public > protected > default > private

### 생성자 (Constructor)
```java
class Student {
    String name;
    Student(String name) {  // 생성자: 클래스명과 동일, 반환 타입 없음
        this.name = name;
    }
}
Student s = new Student("Kim");  // 객체 생성 시 생성자 호출
```

### 상속 (Inheritance)
```java
class Animal { void sound() { System.out.println("..."); } }
class Dog extends Animal {
    void sound() { System.out.println("멍멍"); }  // 오버라이딩
}
```

### 오버로딩 vs 오버라이딩

| 구분 | 오버로딩 | 오버라이딩 |
| :--- | :--- | :--- |
| **메서드명** | 동일 | 동일 |
| **매개변수** | **다름** | 동일 |
| **리턴 타입** | 관계없음 | 동일 |
| **적용 범위** | 같은 클래스 | 상속 관계 |

### 추상 클래스 vs 인터페이스 ★빈출★

| 구분 | 추상 클래스 (abstract class) | 인터페이스 (interface) |
| :--- | :--- | :--- |
| **선언** | `abstract class A { }` | `interface A { }` |
| **구현** | `extends` (단일 상속) | `implements` (다중 구현 가능) |
| **메서드** | 일반 메서드 + 추상 메서드 혼합 가능 | 추상 메서드만 (Java 8부터 default 메서드 가능) |
| **변수** | 다양한 변수 가능 | 상수(public static final)만 |

## ★ 핵심 암기 포인트 ★

1. **전위(++x)=먼저 증가, 후위(x++)=나중에 증가**
2. **배열과 포인터:** `arr[i] == *(arr+i)`, 배열명 = 시작 주소
3. **연산자 우선순위:** 산술 > 시프트 > 관계 > 비트 > 논리 > 대입
4. **변수 스코프:** 지역(함수 내), 전역(프로그램 전체), static(값 유지/함수 내 접근)
5. **접근 제어자:** public > protected > default > private
6. **추상 클래스=extends/단일 상속, 인터페이스=implements/다중 구현**
7. **파이썬 자료형:** 리스트[변경O], 튜플(변경X), 세트{순서X/중복X}, 딕셔너리{키:값}
8. **슬라이싱:** `a[시작:끝:간격]`, `a[::-1]`=역순
9. **printf 서식:** %d(정수), %f(실수), %c(문자), %s(문자열), %o(8진), %x(16진)
10. **재귀함수:** 반드시 종료 조건(Base Case) 확인 → 호출 스택 추적

#### 📌 상속
```java
class Animal{
    private String name;
    public void setName(String name){
        this.name = name;
    }
    public void speak(){
        System.out.println(name + "이(가) 소리를 냅니다.");
    }
}

class Dog extends Animal{
    public Dog(String name){
        super(name);
    }
    public void speak(){
        System.out.println("멍멍!");            // 오버라이딩
    }
}

class Cat extends Animal{
    public Cat(String name){
        super(name);
    }
    public void speak(){
        System.out.println("야옹!");            // 오버라이딩
    }
}

public class Main {
    public static void main(String[] args) {
        Animal animal = new Animal();
        Dog dog = new Dog();
        Cat cat = new Cat();

        animal.setName("동물");
        dog.setName("멍멍이");
        cat.setName("야옹이");

        animal.speak();
        dog.speak();
        cat.speak();
    }
}
```

#### 📌 인터페이스
```java
interface Animal{
    public void speak();
    public static void sleep(){
        System.out.println("잠을 잡니다.");
    }
}

class Dog implements Animal{
    public void speak(){
        System.out.println("멍멍!");
    }
    public static void sleep(){
        System.out.println("강아지가 잠을 잡니다.");
    }
}

class Cat implements Animal{
    public void speak(){
        System.out.println("야옹!");
    }
    public static void sleep(){
        System.out.println("고양이가 잠을 잡니다.");
    }
}

public class Main {
    public static void main(String[] args) {
        Animal dog = new Dog();         // 다형성
        Animal cat = new Cat();

        // 메서드 사용
        dog.speak();
        cat.speak();
        
        // static 메서드의 사용
        Animal.sleep();
        Dog.sleep();
        Cat.sleep();
    }
}
```

#### 📌 C언어 입출력
```c
#include <stdio.h>

int main(){
    int num;
    printf("정수를 입력하세요: ");
    scanf("%d", &num);
    
    printf("%d + %d = %d\n", num, num, num + num);
    printf("%d - %d = %d\n", num, num, num - num);
    printf("%d * %d = %d\n", num, num, num * num);
    printf("%d / %d = %d\n", num, num, num / num);
    printf("%d %% %d = %d\n", num, num, num % num);
    printf("%d\n", ++num);
    printf("%d\n", num++);
    printf("%d\n", --num);
    printf("%d\n", num--);

    return 0;
}
```
